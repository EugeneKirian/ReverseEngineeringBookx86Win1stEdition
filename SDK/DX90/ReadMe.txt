Instructions:

1. Download DirectX 9.0 SDK from https://www.microsoft.com/en-us/download/details.aspx?id=6812
2. Install the SDK to any location on your computer, for example into C:\Program Files (x86)\Microsoft DirectX SDK (June 2010)
3. Open the directory with the installed DirectX SDK.
4. Copy directories Lib and Include into the current directory.
